Sem foto cadastrada, o sistema procura automaticamente o arquivo <usuario>.jpg na raiz do repositório do GitHub (ex: jairo.jpg, ao lado do index.html). Se o arquivo não existir, aparece um avatar colorido com as iniciais. Cada jogador também pode enviar a própria foto direto pelo celular, no botão "Alterar minha foto" dentro da sua área de jogador.

Fotos enviadas pelo app ficam salvas comprimidas dentro do banco de dados. Use com moderação — evite que todo mundo troque de foto toda hora, pra não pesar demais o banco.
